📏 กล้องวัดระยะทาง

แอปพลิเคชันสำหรับวัดระยะทางวัตถุด้วยกล้อง

✨ วัดระยะทางง่ายๆ ด้วยกล้อง
🎯 เล็งแม่นยำด้วยกากบาท
📱 รองรับทั้งมือถือและคอมพิวเตอร์
🌟 แสดงผลเป็นเมตรและเซนติเมตร

⚠️ แอปนี้ต้องการอนุญาตเข้าถึงกล้อง
กรุณากดปุ่ม "อนุญาต" เมื่อ browser ถามสิทธิ์

❌ ไม่สามารถเข้าถึงกล้องได้

กรุณาตรวจสอบ:

✓ คุณได้กดปุ่ม "อนุญาต" เมื่อ browser ขอสิทธิ์หรือไม่

✓ Browser รองรับการเข้าถึงกล้อง (Chrome, Safari, Edge)

✓ กล้องไม่ถูกใช้งานโดยแอปอื่น

✓ เว็บไซต์ต้องรันผ่าน HTTPS หรือ localhost

วิธีแก้ไข:

1. ลองรีเฟรชหน้าเว็บ แล้วกดอนุญาตอีกครั้ง

2. ตรวจสอบการตั้งค่าสิทธิ์กล้องใน browser

3. ลองใช้ browser อื่น

📏 วิธีใช้งาน:
1. เล็งกล้องไปที่วัตถุ
2. กดปุ่มวัดครั้งแรก (จุดเริ่มต้น)
3. เคลื่อนที่ไปยังจุดสิ้นสุด
4. กดปุ่มวัดอีกครั้ง
5. ดูระยะทางที่วัดได้
